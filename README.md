 # CarryOn - スマート持ち物チェックリスト

CarryOnは、旅行、出張、通学、ジムなど、様々なシチュエーションに合わせて持ち物を管理できる、シンプルで直感的なチェックリストアプリです。

## 🌟 主な機能

- **シーン別テンプレート**: 学校、仕事、海外旅行、国内旅行など、標準的な持ち物リストを即座に利用可能。
- **自由なカスタマイズ**: 既存リストの項目追加・削除はもちろん、自分だけのオリジナルリストを「その他」から無限に作成可能。
- **進捗の可視化**: 保存済みリスト画面で、各リストの達成度をプログレスバーで一目で確認。
- **マルチユーザー対応**: シンプルなユーザー登録・ログイン機能により、ユーザーごとにリストを個別に管理。
- **セキュアなストレージ**: 同一ドメイン内の他アプリとの干渉を防ぐ、名前空間（Prefix）付きのLocalStorage運用。

## 🛠 技術スタック

- **Frontend**: HTML5, Vanilla CSS (独自設計), JavaScript (ES6+ Modules)
- **Storage**: Browser LocalStorage / SessionStorage

## 🔐 セキュリティと設計上の配慮

- **キーの衝突防止**: すべてのストレージキーには `carryOn_` プレフィックスが付与されており、GitHub Pagesなどの共有ドメイン下でも安全に動作します。
- **エラーハンドリング**: ユーザー名の重複やパスワードの誤りに対し、明確なフィードバックを画面上に表示します。
- **ユニークID発行**: 「その他」リストの作成ごとにタイムスタンプベースのIDを発行し、データの重複保存や上書きを防止しています。
